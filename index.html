    • EC2 Version Checker

    Project maintained by Alex Luke Hosted on GitHub Pages — Theme by mattgraham

    This is a fabric script to check the release version of Linux servers running in Amazon EC2. On Debian based servers, it will also report the number pending updates.

    Metrics reported

    • Instance ID
    • Server name (from the EC2 Name tag)
    • IP address
    • Public hostname
    • OS version
    • Kernel version
    • Processor architecture
    • Any pending updates
    • Free space on root partition

    Installing

    Install the requirements with pip install -r requirements.txt. Using a virtualenv is recommended.

    Set up your AWS credentials per the Boto docs

    Running

    Running fab in the source directory will fetch a list of servers from EC2 and output a spreadsheet with the above metrics.

    You can specific the username to use with the -u command line option and/or the ssh key file to use with -i.
